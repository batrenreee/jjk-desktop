KENDİ ARKA PLAN MÜZİĞİN
========================

Lanet Düellosu oyunu varsayılan olarak telifsiz, kendi ürettiği
sentezleyici (chiptune) bir müzik çalar.

Kendi şarkını çalmak istersen:
1. İstediğin müzik dosyasını "bgm.mp3" olarak yeniden adlandır.
2. Bu klasörün (music/) içine koy:  music/bgm.mp3
3. Oyunu yeniden derle:  npm run tauri build
   (veya geliştirme modunda:  npm run tauri dev)

Oyun açıldığında bgm.mp3 varsa otomatik olarak onu çalar,
yoksa varsayılan sentez müziğe döner.

NOT (telif): Lady Gaga - Judas gibi telifli şarkıların orijinal
kayıtları yasal olarak uygulamaya gömülemez. Yalnızca sahibi
olduğun / kullanım hakkın olan dosyaları buraya ekle.
